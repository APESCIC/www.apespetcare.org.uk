# APES Pet Care Website

This repository is used to support the ongoing development, maintenance and improvement of the APES Pet Care website:

**Website:** [www.apespetcare.org.uk](https://www.apespetcare.org.uk)

## Purpose of This Repository

This repository helps us track, manage and document development work and fixes relating to the APES Pet Care website. It provides a central place for recording website issues, proposed improvements, technical changes and completed updates.

The repository may be used for:

* Tracking bugs, faults and technical issues
* Recording website improvements and feature requests
* Managing updates to website content, layout and functionality
* Documenting fixes and development decisions
* Coordinating development tasks and priorities
* Maintaining a clear record of website changes over time

## About APES Pet Care

APES Pet Care is part of the wider APES digital infrastructure used to support animal welfare, pet care services, public information and organisational operations.

The website is intended to provide clear, accessible and reliable information for users accessing APES Pet Care services and related resources.

## Issue Tracking

Issues should be used to report and manage website-related work.

When creating an issue, please include as much relevant detail as possible, such as:

* The page or section affected
* A clear description of the issue or requested change
* Steps to reproduce the issue, where applicable
* Screenshots or examples, if useful
* The expected outcome
* Any relevant device, browser or accessibility information
* Priority or urgency, where known

## Development and Fixes

Development work and fixes should be tracked through GitHub issues, branches and pull requests where appropriate.

Each change should aim to:

* Improve website reliability
* Maintain a professional and accessible user experience
* Support clear public communication
* Protect organisational credibility
* Reduce technical risk
* Keep website content accurate and up to date

## Website Areas Covered

This repository may include work relating to:

* Public website pages
* Website content updates
* User experience improvements
* Accessibility improvements
* Mobile and desktop display issues
* Forms and contact routes
* Navigation and layout
* Performance and loading issues
* Broken links or missing content
* Security and maintenance tasks
* Search engine and metadata improvements
